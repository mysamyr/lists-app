! - опис вигляду сторінок і взаємодії між ними (UI)
_ - коментар від мене

Програма "Lister" дає змогу користувачеві керувати власними списками (lists).
Кожен список має свою назву, конфігурацію та інші елементи списку (children для списків) або ж деталі (для елементу списку (list-item)).
Назвою є строка довжиною від 3 до 30 символів.
Конфігурацію являє собою список правил для елементів списку (кількість полів в елементі списку, довжина/величина, тип цих полів, шаблон відображення, префікси, постфікси).

Кожен список (list) та елемент списку (list-item) може бути створений,
його назва може бути змінена на іншу (унікальну серед інших списків одного рівня) і видалений. Список створюється пустим (без елементів).

! При потраплянні користувача на головну сторінку (home) він бачить:
- хедер з лише назвою програми (Lister App),
- хлібні крихти (breadcrumbs з вказівником "Home"),
- список всіх списків першого рівня,
- кнопку для створення нового списку.

Кожний список (list) містить назву та кнопки для перейменування і видалення.
При натисканні на список, користувач переходить на сторінку з його елементами (іншими списками або елементами списку).
При натисканні на кнопку створення, з'являється випадайка.
При натисканні на кнопку зміни назви, користувачеві відкривається модальне вікно з полем для вводу (input),
яке містить вже введену назву і 2 кнопки для підтвердження зміни назви чи закриття модального вікна без змін.
При натисканні на кнопку видалення, користувачеві відкривається модальне вікно з кнопками для підтвердження видалення чи закриття модального вікна без видалення.

Будь-яке модальне вікно також може бути закрите натисканням за його межі або натисканням кнопок Escape чи Tab з клавіатури.

! При потраплянні користувача на сторінку списку (list-page), користувач бачить:
- хедер з кнопками Back і Sort,
- хлібні крихти (зі списком всіх списків, які довелося пройти, щоб добратися до даного списку, починаючи з Home),
- список всіх списків чи елементів батьківського списку,
- кнопку для створення нового списку чи елементу списку.

Елемент списку (list-item) відрізняється від списку тільки кольором заднього фону (список - сірий, елемент списку - білий).
